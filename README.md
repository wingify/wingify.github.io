### Local Setup

**Requirements**

1. Node.js `v12+`
2. Yarn `v1.22+`

**Steps:**

1. Clone the repository

```bash
git clone https://github.com/wingify/wingify.github.io.git
```

2. Install the dependencies

```bash
yarn install
```

3. Build and start the local server

```bash
yarn develop
```

Open [http://localhost:8000/](http://localhost:8000/)
